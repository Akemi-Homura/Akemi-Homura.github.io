Manthan, Codefest 16部分题解

A题

题目大意

有两种枪，第一种枪每发子弹造成$a$点伤害，第二种枪每发子弹造成$b$点伤害，问能否恰好造成$c$点伤害。

算法

题目的本质是求方程$ax+by=c$的整数解，可以用拓展欧几里德，不过这道题数据范围较小，可以暴力枚举$x$的值，再看$y$是否是整数。

代码

B题

题意大意

问有多少个$n$满足$n!$后面有$m$个$0$。

算法

因为$m \leq 10^5$，所以可以暴力枚举$[1,m]$,分别计算每个数的阶乘有多少个$0$，满足答案的输出即可。

计算$n!$有几个$0$的方法：$sum=\displaystyle{\sum_{i=1}^{log_5 n}\left[\frac{n}{5^i}\right]}$。

D题

题目大意

有$n$个数，要求重新调整这$n$个数的顺序，使得前缀满足$f_{n+2}=f_{n+1}+f_{n}$的序列最长。

算法

先来简单的分析一下。如果我们知道了这个序列的前两项那么整个序列就确定了，因此可以枚举前两项，然后看这个序列最多能有多长，唯一的优化是记录哪些前两项被枚举过，防止被0 0 0 0…这样的数据$\rm Hack$，其他的优化就不必做了，乱优化反而会弄巧成拙。

可以暴力的原因是最多只有几个序列的长度接近$n$，其他的序列都很短，所以总的复杂度远小于$n^3$，加上$\rm cf$神机，完全可以跑过。
